필수 도구 설치 가이드

  • Git: 터미널에서 brew install git 실행 또는 Xcode Command Line Tools 자동 설치 팝업 확인
  • Maven (mvn): 터미널에서 brew install maven 실행
  • Node.js / npm: 터미널에서 brew install node 실행
  • Docker: Docker Desktop for Mac 설치
  • Git: Git 공식 사이트에서 다운로드 혹은 winget install --id Git.Git 실행
  • Maven (mvn): Maven 공식 사이트 다운로드 혹은 winget install --id Apache.Maven 실행
  • Node.js / npm: Node.js 공식 사이트 다운로드 혹은 winget install --id OpenJS.NodeJS 실행
  • Docker: Docker Desktop for Windows 설치 (WSL2 필수)

범례 (Legend)

Tier — 실행 난이도/성격
  • Tier 1 즉시 기동 — 추가 인프라 없이 바로 실행 (예: boot-sample, simple-react)
  • Tier 2 빌드/라이브러리 — 띄울 화면 없이 빌드·설치만 (예: runtime)
  • Tier 3 인프라 필요 — Docker·Ollama 등 사전 준비 필요 (예: msa, ai-rag)
DeployType — 기동 방식
  • boot Spring Boot 내장 톰캣 (▶ Run) · war 외부 Tomcat 배포 (▶ Tomcat 기동) · react Vite 개발서버 (npm) · lib 빌드 전용
상태(Status) 배지
  • idle 아직 시작 안 함 (소스 없음)
  • cloned 소스 준비됨 (클론 완료, 실행 전)
  • cloning 소스 내려받는 중
  • building 빌드 중 (mvn/npm)
  • running 실행 중 (포트 표시)
  • done 완료 (클론/빌드 완료 또는 정상 종료)
  • stopped 사용자가 중지함
  • error 오류 발생 (로그 확인)
